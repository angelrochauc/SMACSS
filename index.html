Metodología SMACSS

Organizando CSS

Las Metodologías o Arquitecturas CSS nos ayudaran a escribir código CSS más predecible, reutilizable, mantenible y escalable.


A continuación, un breve resumen de la metodología SMACSS la cual es una de las más usadas para organizar nuestros estilos CSS:


SMACSS (pronunciado “smacks”) es la sigla de Scalable and Modular Architecture for CSS y es una metodología CSS que tiene como objetivo que el código CSS sea mejor, más fácil de leer y modular.


Bootstrap, Fundation y muchos otros frameworks se basan en SMACSS para definir sus estructuras.


De acuerdo con el autor de la metodología SMACSS, Jonathan Snook, el considera su documento como una guía, más que algo estricto para seguir al pie de la letra. En palabras de Snook:


“Es una guía de estilo más que un marco rígido. Es una forma de examinar su proceso de diseño y una forma de encajar esos marcos rígidos en un proceso de pensamiento flexible. Es un intento de documentar un enfoque consistente para el desarrollo del sitio cuando se usa CSS."


En alguna parte de su documento, Jonathan Snook, enfatiza: "Siéntase libre de utilizar esto en su totalidad o solo las partes que funcionen mejor para usted. O de no usarlo en absoluto. Cuando se trata de desarrollo web, la respuesta a la mayoría de las preguntas es: depende".
